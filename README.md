# ⚗️ FormulaX — L'Atlas des Sciences

> Toutes les formules essentielles en Mathématiques, Physique et Chimie, du primaire aux études supérieures.

![Preview](https://img.shields.io/badge/HTML-5-orange) ![CSS](https://img.shields.io/badge/CSS-3-blue) ![JS](https://img.shields.io/badge/JavaScript-ES6-yellow) ![MathJax](https://img.shields.io/badge/MathJax-3.2-green)

## 🚀 Voir le site

Ouvrez simplement `index.html` dans votre navigateur, ou hébergez sur GitHub Pages.

## 📚 Contenu

### Mathématiques
- 🟢 **Primaire** : Opérations, périmètre, aire
- 🟡 **Collège** : Identités remarquables, Pythagore, proportionnalité
- 🟠 **Lycée** : Équation du 2nd degré, trigonométrie, dérivation, intégration

### Physique
- 🟡 **Collège** : Vitesse, loi d'Ohm
- 🟠 **Lycée** : Lois de Newton, énergie, thermodynamique, relativité (E=mc²), ondes

### Chimie
- 🟡 **Collège** : Structure de l'atome
- 🟠 **Lycée** : pH, stœchiométrie, cinétique, oxydo-réduction

### Exercices corrigés
6 exercices avec solutions détaillées étape par étape.

### Constantes fondamentales
c, g, h, Nₐ, R, F, π, e

## 🛠️ Technologies

- **HTML5** — Structure sémantique
- **CSS3** — Variables, animations, responsive, dark/light mode
- **JavaScript ES6** — Interactivité, scroll animations
- **MathJax 3** — Rendu LaTeX des formules
- **SVG** — Schémas dessinés à la main

## ✨ Fonctionnalités

- 🌙 Mode sombre / ☀️ Mode clair (bascule + sauvegarde locale)
- 📐 Formules en LaTeX avec rendu mathématique
- 🎨 Schémas SVG interactifs et colorés
- ✏️ Exercices avec solutions masquées/révélées
- 📱 Responsive mobile
- 🖨️ Styles d'impression
- 🔄 Animations d'apparition au scroll

## 🌍 Déploiement GitHub Pages

1. Fork ce repository
2. `Settings` → `Pages` → `Source: main / root`
3. Le site sera disponible à `https://username.github.io/formulax`
